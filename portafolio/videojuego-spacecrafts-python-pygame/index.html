Descripción del proyecto

Spacecrafts es un videojuego arcade 2D desarrollado en Python utilizando la librería Pygame, creado para demostrar la implementación de las principales mecánicas de desarrollo de videojuegos. El proyecto presenta un nivel donde el jugador controla una nave espacial para esquivar ataques enemigos y eliminarlos mediante disparos.

El juego incorpora un sistema de puntuación, contador de vidas, detección de colisiones y ventanas interactivas de bienvenida, pausa, victoria y game over, generación de enemigos y gestión de estados de la partida. El resultado es un proyecto funcional y entretenido que evidencia competencias en programación orientada a objetos, lógica de videojuegos y desarrollo interactivo con Python.

Aunque este proyecto no está relacionado con el análisis de datos, que es el área en el que me desempeño actualmente, se incluye en el portafolio como una muestra de mis competencias en desarrollo de software. Spacecrafts fue creado originalmente hace 15 años utilizando Python 2; sin embargo, decidí retomarlo para migrarlo a una versión actual del lenguaje, rediseñar su arquitectura y aplicar mejores prácticas de programación.

Sobre el entorno

El proyecto fue desarrollado en Python 3.11 utilizando la librería Pygame 2.6.1 como motor principal para la gestión de gráficos, eventos, sonido y detección de colisiones. La aplicación se implementó sobre un entorno virtual (venv), siguiendo buenas prácticas de organización del código mediante programación orientada a objetos, separación de responsabilidades y estructura modular, lo que favorece la escalabilidad y el mantenimiento del proyecto.
